Detailed Description

A ProfileSimulator is a random walk simulator based on the colision profile. There's an open possibiliy to implement other kinds of simulators (for example, using other kinds of distribution, such as the maximal ball, or a decoupled profile). In order to create a new kind of simulator, it must be inherited from the class Simulator.
