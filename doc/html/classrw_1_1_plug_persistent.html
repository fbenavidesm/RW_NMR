Detailed Description

This class handles all relevant information about a Random Walk simulation. This information can be stored in a file or can be used to populate a rw::Formation. The walkers position, the RND seed, the last position of walkers, the Laplace transform and other simulation values are stored with this class. All simulation parameters can be stored in file and can be used to reproduce the walker's path for a genetic algorithm optimization.
