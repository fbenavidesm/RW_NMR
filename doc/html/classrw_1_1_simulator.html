Detailed Description

A Simulator is a class that mimics a random walk simulation without using a texture (image). It uses a basis function or a profile that groups the main characteristics of a real random walk. A Simulator executes a random walk based on this profile, distributing uniformly the walker's probability to hit. The walker's probability to hit is called XI (Colision rate) and it can be stable or changing along time, when the walker moves between different size pores.
