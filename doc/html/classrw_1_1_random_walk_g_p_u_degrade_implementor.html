Detailed Description

This walker handling parallelizes the random walk movement in the GPU. It is important to note that this particular walker handling demands several rules:

1) The number of walkers must be a multiple of 1000.

2) If the number of walkers is more than 128 000, then they wll have to be split in several groups and the gain in performance is less noticeable.
