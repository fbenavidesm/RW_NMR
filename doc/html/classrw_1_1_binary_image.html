Detailed Description

An encapsulated method to store large binary images, using only one bit per pixel. For example, a 1000x1000x1000 image will require only 120 MB to be stored, which faciliates handling these large textures even in a modest GPU. This class can also handle fast morphologic operations such as Dilation and Erosion by spherical structuring elements. The opening stores a sequence of opened images which can be useful to estimate a pore size distribution.
